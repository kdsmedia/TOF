# ToF App

This is a turn-based strategy game project.

## About the Project

This repository contains the full source code for "ToF App", a classic turn-based strategy game. The project is built using the Godot Engine.

## Building from Source

To build this project, you will need **Godot Engine version 2.1.6**.

1.  Download Godot 2.1.6 from the official archives.
2.  Open the Godot editor.
3.  Select "Import Project" and browse to this project's directory.
4.  Choose the `engine.cfg` file to open the project.
5.  From the Godot editor, you can run the game or use the "Export" menu to build it for your desired platform.
